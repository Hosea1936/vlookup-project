# vlookup-project
This project demonstrates how to use the VLOOKUP function in Excel to search for employee number and return their monthly salary,age,education qualification and the city.
Examples of VLOOKUP formula used are =VLOOKUP(L2,A2:I51,9,FALSE), =VLOOKUP(L2,A2:I51,8,FALSE), =VLOOKUP(L2,A2:I51,6,FALSE), =VLOOKUP(L2,A2:I51,5,FALSE)
Skills demonstrated are Data cleaning,VLOOKUP function,Table referencing
